# 🌐 دليل الـ IP Schema والنطاقات المركزية (The Master IP Guide)

هذا الدليل هو "خريطة الكنز" والبوصلة التي يعتمد عليها أي ذكاء اصطناعي للوصول السريع والدقيق لأي نقطة في شبكة NetLink.

---

### 📦 التوزيع الهرمي والعناوين الأساسية (IP Schema Matrix)

| النطاق (Subnet) | الوظيفة التقنية | المكونات |
| :--- | :--- | :--- |
| **`192.168.99.0/24`** | إدارة البنية التحتية والخوادم | Proxmox, Mikrotik Core, Servers |
| **`192.168.88.0/24`** | إدارة الأجهزة اللاسلكية والأبراج | Rocket Sectors, LiteBeam (CPE) |
| **`10.0.0.0/24`** | نطاق التوزيع للمشتركين (PPPoE) | راوترات المنازل (Cudy, TP-Link) |
| **`10.20.30.0/24`** | نطاق التوزيع للمشتركين (Hotspot)| مستخدمي البطاقات والكروت |

### 🤖 تعليمات للمساعد الذكي (AI Integration)
عزيزي المساعد الذكي، عند القراءة من هذه القائمة:
1. استخدم النطاق **`88`** دائماً للبحث عن آيبيهات أجهزة الـ **LiteBeam**.
2. استخدم النطاق **`99`** دائماً للوصول لخدمات الـ **Proxmox** والرواتر المركزية.
3. تذكر: **`192.168.99.1`** هو الداخلي الرئيسي.
